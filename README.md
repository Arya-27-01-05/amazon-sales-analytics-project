# Amazon Sales Analytics & Customer Insights Project

## Project Overview

This project analyzes Amazon sales data using SQL and Power BI to identify revenue trends, customer behavior, product performance, and regional profitability.

## Tools Used

* SQL
* Excel
* Power BI

## Files Included

* Amazon Sales Dashboard.pbix
* Amazon sales.csv
* sql project.sql
* Sql_project_report.docx

## Analysis Performed

* Revenue Analysis
* Regional Profitability Analysis
* Customer Analysis
* Product Performance Analysis
* KPI Calculations

## KPIs

* Total Revenue
* Total Profit
* Total Orders
* Average Order Value

## Key Insights

* Some regions generated negative profits.
* High-value customers contributed significantly to revenue.
* Certain product categories generated lower sales.
* The business processed 3,099 orders.
* Average Order Value was approximately 230.23.

## Business Recommendations

* Reduce excessive discounting in low-profit regions.
* Focus on high-performing products and customers.
* Improve customer retention strategies.
* Monitor regional profitability regularly.
